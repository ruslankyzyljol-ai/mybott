# PROFIX — Telegram Bot

## Орнотуу

### 1. config.py файлын ачып толтуруңуз:
- BOT_TOKEN — BotFather дан алынган токен
- ADMIN_ID — Өз Telegram ID ңиз (@userinfobot дан алса болот)
- FOOTBALL_API — football-data.org дан бекер алынат
- CHANNEL_ID — Каналыңыздын username си (@channel_name)

### 2. Букмекер ссылкаларын коюңуз:
config.py дагы BOOKMAKERS бөлүмүн толтуруңуз

### 3. Railway де иштетүү:
1. github.com да репо жасаңыз
2. railway.app га кириңиз
3. GitHub репону тандаңыз
4. Deploy басыңыз — бүттү!
